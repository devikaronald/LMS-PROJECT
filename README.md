# 🎓 Learning Management System (LMS)

A full-stack Learning Management System (LMS) built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). This application provides a modern platform for students to enroll in courses, access learning materials, and track their progress, while instructors can create and manage courses through a dedicated dashboard.

## ✨ Features

- 🔐 JWT Authentication
- 👨‍🎓 Student & Instructor Role-Based Access
- 📚 Course Management (Create, Update, Delete)
- 🎥 Video Lesson Support
- 📝 Student Enrollment
- 📈 Learning Progress Tracking
- 📱 Responsive User Interface
- 🌐 RESTful API Integration

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas, Mongoose
- **Authentication:** JWT, bcrypt
- **Deployment:** Vercel (Frontend), Render (Backend)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/devikaronald/LMS-PROJECT.git
```

### Install dependencies

```bash
# Frontend
cd frontend
npm install

# Backend
cd ../backend
npm install
```

### Run the project

```bash
# Frontend
npm run dev

# Backend
npm start
```
